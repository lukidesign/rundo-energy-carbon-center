## 🔴 A. 性能优化（立竿见影）

| # | 优化项 | 现状 → 目标 | 工作量 |
|---|--------|------------|--------|
| A1 | **React 切换 production 版本** | 当前用 `react.development.js`（含运行时警告检查、更大体积），→ 换 `react.production.min.js` + `react-dom.production.min.js`，体积减半、性能提升 | ⚡极小 |
| A2 | **FloorView3D 帧跳过 bug 修复** | Line ~1058：跳帧时仍然执行 `renderer.render()`，浪费 GPU → 改为 `return` 直接跳过整帧 | ⚡极小 |
| A3 | **Globe 球体 LOD 降低** | 地球 `SphereGeometry(1, 64, 64)` = 8,192 个三角面，用户实际看不出 → 改 `(1, 48, 48)` 即可，减 44% 面数 | ⚡极小 |
| A4 | **图片资源空闲预加载** | 切换楼层时 `zbuilding3dimg/` 首次加载白屏 → 页面 idle 后用 `new Image()` 批量预加载 7 层图片 | 小 |
| A5 | **Intro 视频快速跳过** | 当前强制 8 秒或手动跳过 → 改为 2-3 秒后自动淡入主界面，视频继续作为背景播放 | 小 |
| A6 | **Chart.js 首次渲染延迟去除** | EnergyChart 首次 mount 有 2100ms delay、SavingsChart 有 2050ms → 可缩短至 500ms，让数据更快可见 | 小 |

## 🔵 D. 代码质量与健壮性

| # | 优化项 | 现状 → 目标 | 工作量 |
|---|--------|------------|--------|
| D1 | **关键按钮 aria-label** | 所有按钮无语义标签，屏幕阅读器无法识别 → 为 Sidebar、MapControls、Drawer 关闭按钮添加 `aria-label` | 小 |

| D4 | **Chart.js 注册全局化** | 当前每个 Chart 组件内可能重复注册插件 → 在顶层统一 `Chart.register(...)` 一次 | ⚡极小 |
